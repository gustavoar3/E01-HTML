Tutorial Sobre Estruturas de Repetição

Loop While

O loop while mantém uma repetição enquanto uma condição for verdadeira.

Sua estrutura é:

while( condição ){ código }

Loop For

Já o loop for mantém um loop n vezes, apartir do valor de uma variavel.

Sua estrutura é:

for( variavel = valor; variavel < n; variavel++ ){ código }

Exemplos

While

int inteiro = 0; while(inteiro < 100){ printf("%d", inteiro); inteiro++; }

For

for(int i = 0 ; i < 100; i++){ printf("%d", i); }
